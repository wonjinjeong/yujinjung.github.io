# Direct3D 12의 개요

## Direct3D
- Microsoft의 DirectX API에서 3D Graphics 연산과 출력을 담당하는 부분
- 응용 프로그램에서 GPU를 제어하고 프로그래밍하는데 쓰이는 저수준 그래픽 API
- **Direct3D라는 간접층과 HW Driver**가 응용 프로그램과 그래픽 하드웨어 사이에 서 Direct3D 명령들을 시스템의 GPU가 이해하는 기계어로 번역한다.
