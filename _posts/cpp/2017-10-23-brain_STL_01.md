# 함수포인터
## 함수포인터란
- 함수의 주소를 저장하는 변수
- 함수의 시작 주소를 저장하는 포인터

### 포인터
```cpp
#include <iostream>
using namespace std;

int main()
{
    int n = 10;     // 정수형 변수
    int *pn = &n;   // 포인터 변수

    return 0;
}
```

### 함수포인터의 선언과 사용
```cpp
#include <iostream>
using namespace std;

void Print(int n)
{
    cout << "정수: " << n << endl;
}

int main()
{
    // void Print(int n)의 함수 포인터 선언
    void (*pf)(int);
    //함수의 이름은 함수의 시작 주소
    pf = Print;

    Print(10);  // 함수 호출
    pf(10);     // 포인터를 이용한 함수 호출
    (*pf)(10);  // 포인터를 이용한 함수 호출

    cout << endl;
    cout << Print << endl;
    
}

```
